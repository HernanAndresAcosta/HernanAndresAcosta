## 🚀 Proyecto Destacado

### 🔗 [CI/CD Multi-Environment DevOps Pipeline](https://github.com/hernan130/desafio-7)

Pipeline CI/CD completo que automatiza el despliegue de infraestructura en tres entornos
(**dev**, **staging** y **producción**) usando ramas de Git como disparador automático.

> Este proyecto es el trabajo integrador final del Bootcamp DevOps Engineer (150 hs),
> e integra **todas las tecnologías vistas durante la formación**.

| Componente | Tecnología |
|------------|------------|
| Orquestación CI/CD | Jenkins Multibranch Pipeline |
| Configuración de infraestructura | Ansible + Jinja2 |
| Contenedores y orquestación | Docker + Kubernetes (Minikube) |
| Exposición segura | Ngrok + GitHub Webhooks |
| VMs objetivo | Multipass — Ubuntu 24.04 LTS |
| Seguridad | SSH Ed25519 + Jenkins Credential Store |
| Imagen personalizada | Docker Hub: `hernan1305/jenkins-ansible:latest` |

**Flujo completo:** `git push` → webhook → Ngrok → Jenkins → Ansible → deploy en VM ✅
